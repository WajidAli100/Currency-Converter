# Currency Converter

A simple **Python command-line Currency Converter** that allows you to convert between multiple currencies in real-time. Perfect for developers, travelers, and anyone who needs quick currency conversions.

## 🚀 Features

- Convert between multiple currencies
- Real-time exchange rates
- User-friendly command-line interface
- Handles invalid inputs gracefully

## 🛠️ Requirements

- Python 3.x
- Internet connection (for fetching live exchange rates)
- Optional: `requests` module (install via `pip install requests`)

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/WajidAli100/Currency-Converter.git
2. Change directory:
  cd Currency-Converter
3. Install dependencies (if any):
  pip install -r requirements.txt
▶️ How to Use

Run the converter script using Python:
  python currency_converter.py
Follow the on-screen prompts:
[spa](https://oasiqspa.com/services/)
Enter the amount
Enter the source currency (e.g., USD)
Enter the target currency (e.g., EUR)

The script will display the converted amount based on the latest exchange rates.

📌 Example
Enter amount: 100
Enter source currency (e.g., USD): USD
Enter target currency (e.g., EUR): EUR

Converted Amount: 92.50 EUR
⚙️ Supported Currencies

Supports most global currencies like:

USD, EUR, GBP, INR, JPY, AUD, CAD, CHF, CNY, and more.
🧪 Error Handling
Alerts if invalid currency codes are entered
Handles network errors gracefully
